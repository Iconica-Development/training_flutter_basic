---
### Flutter cli
- Most used commands:
    - create 'name'
    - doctor
    - run (-d 'deviceid')

---
### Flutter IDE
- Android Studio, version 2020.3.1 (Arctic Fox) or later
    - Flutter Plugin (run debug, hot reload)
    - Dart Plugin (code validation)
- VS Code, latest version
    - Dart code extension
    - Flutter extension

Validate: Flutter doctor from terminal
